### Task 4: Modbus poll + auto-create catalog

Theo docs/contracts/modbus-map-v1.md và docs/thiet_ke_db.md.

1. ModbusMapParser (header-only): header HR0–HR6 (Na, Ndi, Ndo); analog blocks ABCD; unpack FC02/FC01 bit arrays
2. ModbusService: QThread worker, QModbusTcpClient hỗ trợ FC 01, 02, 03 (ADR ngắn nếu chọn QTcpSocket thay thế)
3. ModbusBridge mỗi chu kỳ: FC03 header + analog chunks (multi-PDU nếu Na>14) → FC02 (Ndi) → FC01 (Ndo)

   - QueuedConnection về UI thread
